# new-concept-english

新概念学习平台，目前还在开发中。

## 资源包

平台使用 next 构建，目标是构建出 SSG 托管到 GitHub 的 pages 上。

使用 AI 处理了一些新概念的课本文件，输出成了 json，以及从其他开源仓库拿取了音频和 pdf 等文件。

如果你想使用我已经处理好的资源包，可以[点击查看](https://github.com/new-concept-english/source)。
